# Python课程结课作业：在线记事本系统
提交人：yinying
班级：软件24-2

## 文件说明
1. main.py：记事本核心增查逻辑
2. app.py：Flask网页服务入口，对接学校API接口
3. templates/index.html：前端展示页面
4. .env：存储学校API密钥配置

## 功能介绍
1. 网页新增、查看笔记
2. 校园网环境读取API Key，调用校内接口
3. GitHub版本管理，支持代码拉取、推送
4. Railway云平台部署公网访问

## 本地运行
1. 安装依赖：`pip install -r requirements.txt`
2. 修改.env填入学校真实API密钥
3. 启动项目：`python app.py`
